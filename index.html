Past Projects

Terrain Renderer

A terrain rendering engine written from scratch in C++ using GLFW and OpenGL. The terrain is seperated into chunks which switch to a lower LOD based on how distant they are. The shaders include correct normal mapping, texture blending, as well as cascading shadow maps. The terrain is procedurally generated on the fly as the camera moves through the world and automatically deleted when too far away. The generation is done using perlin noise from FastNoise2, a custom encoded node tree generator and seed can be inputted in the seed.txt file. WASD controls, 1 and 2 can be pressed to switch between wireframe and normal mode.

A simple endless runner game, avoid cars and see how far you can make it. Rendering is done from scratch using C++ and OpenGL. Made for a 2nd year university project. Use A and D to steer and W and S to speed up and slow down.
